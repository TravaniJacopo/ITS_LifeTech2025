# ITS_LifeTech2025
# 📁 Progetti ITS LifeTech 2025/2026

**Autore:** Jacopo Travani

Questa cartella raccoglie tutti i progetti sviluppati durante il percorso formativo presso ITS LifeTech nell’anno accademico **2025/2026**.
I lavori sono suddivisi per area tecnologica, riflettendo le competenze acquisite nel corso dell’anno.

---

## 🔹 Front-End

In questa sezione sono presenti i progetti dedicati allo sviluppo di interfacce web moderne e responsive.
Le tecnologie utilizzate includono:

* **Angular** per la creazione di applicazioni strutturate e dinamiche
* **Bootstrap** per il design responsive e la gestione del layout
* **HTML, CSS e JavaScript** per la costruzione e personalizzazione delle interfacce

Questi progetti dimostrano competenze nella progettazione UI, nella gestione dei componenti e nell’interazione utente.

---

## 🔹 Java OOP

Questa sezione contiene progetti sviluppati in **Java** seguendo i principi della **programmazione orientata agli oggetti (OOP)**.

Tra i principali lavori è presente un sistema per la **gestione degli accessi a un pronto soccorso**, che include:

* gestione delle code dei pazienti
* classificazione delle priorità
* organizzazione dei flussi di accesso

Il progetto evidenzia l’utilizzo di classi, oggetti, incapsulamento e strutture dati per risolvere problemi reali.

---

## 🔹 Back-End

Qui sono raccolti i progetti dedicati allo sviluppo lato server.
È stato utilizzato:

* **Java Spring (Spring Boot)** per la creazione di API e servizi backend

Le applicazioni sviluppate mostrano:

* gestione delle richieste client-server
* strutturazione di API REST
* connessione con database e logica di business

---

## 📌 Obiettivo della raccolta

Questa repository rappresenta un percorso di crescita tecnica e pratica, con l’obiettivo di consolidare competenze nello sviluppo **full stack**, combinando frontend, logica applicativa e backend.
