# 📁 Progetti ITS LifeTech 2025/2026

Autore: Jacopo Travani

Questa cartella raccoglie tutti i progetti sviluppati durante il percorso formativo presso ITS LifeTech nell’anno accademico 2025/2026. I lavori sono suddivisi per area tecnologica, riflettendo le competenze acquisite nel corso dell’anno.

---

## 🔹 Front-End

In questa sezione sono presenti i progetti dedicati allo sviluppo di interfacce web moderne e responsive. Le tecnologie utilizzate includono:

* Angular per la creazione di applicazioni strutturate e dinamiche
* Bootstrap per il design responsive e la gestione del layout
* HTML, CSS e JavaScript per la costruzione e personalizzazione delle interfacce

Questi progetti dimostrano competenze nella progettazione UI, nella gestione dei componenti e nell’interazione utente.

---

## 🔹 Java OOP

Questa sezione contiene progetti sviluppati in Java seguendo i principi della programmazione orientata agli oggetti (OOP).

Tra i principali lavori è presente un sistema per la gestione degli accessi a un pronto soccorso, che include:

* gestione delle code dei pazienti
* classificazione delle priorità
* organizzazione dei flussi di accesso

Il progetto evidenzia l’utilizzo di classi, oggetti, incapsulamento e strutture dati per risolvere problemi reali.

---

## 🔹 Back-End

Qui sono raccolti i progetti dedicati allo sviluppo lato server. È stato utilizzato:

* Java Spring (Spring Boot) per la creazione di API e servizi backend

Le applicazioni sviluppate mostrano:

* gestione delle richieste client-server
* strutturazione di API REST
* connessione con database e logica di business

---

## 🔹 Networking

Questa sezione raccoglie le esercitazioni e i progetti relativi al networking e alla configurazione delle reti, sviluppati principalmente tramite Cisco Packet Tracer (PKT).

Durante le attività sono stati affrontati diversi argomenti fondamentali delle reti informatiche, tra cui:

* configurazione di reti LAN e WAN
* assegnazione dinamica degli indirizzi IP tramite DHCP
* utilizzo e configurazione dei principali protocolli di routing
* gestione della comunicazione tra reti differenti
* tecniche di traduzione degli indirizzi di rete (NAT)
* configurazione di router, switch e dispositivi di rete
* verifica della connettività e troubleshooting

I progetti dimostrano competenze pratiche nella progettazione e simulazione di infrastrutture di rete, nell’analisi del traffico e nella gestione dei servizi di rete.

---

## 📌 Obiettivo della raccolta

Questa repository rappresenta un percorso di crescita tecnica e pratica, con l’obiettivo di consolidare competenze nello sviluppo full stack e nell’amministrazione di reti, combinando frontend, logica applicativa, backend e infrastrutture di networking.
